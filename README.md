# Installation guide

### Running Server

```
$ git clone https://github.com/syahirdev/food-recipe.git
$ cd server
$ yarn install && yarn develop
```

### Running Client (Android)

```
$ cd client
$ yarn install && yarn start
$ yarn android
```
